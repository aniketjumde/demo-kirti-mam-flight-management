4. FLIGHT ID GENERATION LOGIC
====================================================================

Format: [2 Letters][5 Digits]

Example:
- Flight Name: "Indigo"    → Flight ID: "IN10000"
- Flight Name: "SpiceJet"  → Flight ID: "SP10001"
- Flight Name: "Emirates"  → Flight ID: "EM10002"

Algorithm:
1. Extract first 2 characters from flight name
2. Convert to uppercase
3. Get next value from FlightId_Seq
4. Format sequence value as 5-digit string (with leading zeros)
5. Concatenate: [2 letters] + [5 digits]
